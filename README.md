 ☕ Coffee-Shop ☕
 ========================

Coffee shop project.


**Technologies**
------------------------

- NodeJs
- ExpressJs
- MongoDB
- Mongoose (ODM)
- Cloudinary
- Google Auth Library

# Install

 :warning: First edit the **\_.env ** file for your database. Later:

* install dependencies

npm install

* To run the API use the command.

npm start
